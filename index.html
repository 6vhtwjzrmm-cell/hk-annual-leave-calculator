Hong Kong Employment Ordinance Cap. 41 - Annual Leave Calculation Tool

📋 香港雇佣条例第41章 - 年假天数规定
雇员按连续性合约受雇每满 12 个月，便可享有有薪年假。年假天数按服务年期递增：
服务年期 法定有薪年假
1-2年7天
3年8天
4年9天
5年10天
6年11天
7年12天
8年13天
9年或以上14天
📌 计算逻辑说明（公司指引）
计算分两段：
① 当前年度（如2026）：法定年假 = 上一个日历年（2025全年）累积的法定天数（按正进行的服务年度）
   按比例额外 = (当年法定 - 上年法定) × 本年度工作天数 / 365
② 下一年度（如2027）：按比例法定 = 当年法定天數 × 本年度工作天数 / 365（向上取整）

案例（CUI HAOFAN）：2022/7/12入职，2026/6/12离职，年假14天，已用8天
→ 2026结算：Statutory(2025累积)=9天 + Pro-rata Additional=(10-9)×163/365≈0.5天 → 合计9.5天，已用8天 → Unused 1.5天
→ 2027结算：Pro-rata Statutory=10×163/365≈4.465 → 向上取整=5天 → Unused 5.0天
→ 总计未使用：1.5 + 5.0 = 6.5天
注：输入员工享有的年假总天数（含法定年假及公司额外年假）
注：从「法定+额外」池中已使用天数
注：通常为0（下年度尚未使用）
