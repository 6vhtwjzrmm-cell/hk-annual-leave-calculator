📋 香港雇佣条例第41章 - 年假天数规定
雇员按连续性合约受雇每满 12 个月，便可享有有薪年假。年假天数按服务年期递增：

9年或以上14天
📌 计算逻辑说明（公司指引）
计算分两段：
① 当前年度（如2026）：法定年假 = 上一个日历年（2025全年）累积的法定天数
   按比例额外 = 额外年假 × 本年度工作天数 / 365
② 下一年度（如2027）：按比例法定 = 下一年度法定天數 × 本年度工作天数 / 365

案例（CUI HAOFAN）：2022/7/12入职，2026/6/12离职，年假14天
→ 2026结算：Statutory(2025累积)=9天 + Pro-rata Additional=(14-9)×163/365≈2.5天 → Unused 1.5天
→ 2027结算：Pro-rata Statutory=10×163/365≈5.0天 → Unused 5.0天
→ 总计未使用：6.5天
注：输入员工享有的年假总天数（含法定年假及公司额外年假）
注：从「法定+额外」池中已使用天数
注：通常为0（下年度尚未使用）

⚠️ 香港《僱傭條例》相關條款 (Relevant Employment Ordinance Provisions):
  • 受僱期少於3個月：僱員不享有有薪年假，僱主無須支付年假薪酬。
  • 受僱期滿3個月但未滿12個月：可獲發放按比例計算的年假薪酬（因犯嚴重過失遭即時解僱除外）。
  • 受僱期滿12個月：完整計算 — 上一年度（日曆年）法定 + 本年度按比例額外 + 本年度按比例法定（歸入下年）。
  • 本年度 = 離職日曆年（從1月1日到離職日）
